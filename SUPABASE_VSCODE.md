# Extensiones de Supabase para VS Code

## Instalación recomendada

### 1. Extensión oficial de Supabase
Busca en VS Code Extensions (Ctrl+Shift+X):
- **Supabase** por Supabase

O instala desde línea de comandos:
```bash
code --install-extension supabase.supabase
```

### 2. Conectar con tu proyecto

1. Abre VS Code en la carpeta del proyecto
2. Presiona `Ctrl+Shift+P` 
3. Escribe: `Supabase: Link with Project`
4. Ingresa el Project Ref: `avothswkmrkwedkwymra`

### 3. Configuración manual

Si prefieres configurar manualmente, ya tienes las credenciales en `.env.local`:

- `Supabase: Open SQL Editor` - Abrir editor SQL
